# Desafio LocalizaLabs - Catálogo de Produtos

Este projeto é um catálogo de produtos desenvolvido em React + TypeScript como parte do desafio LocalizaLabs. Ele consome a API pública [EscuelaJS Products](https://api.escuelajs.co/api/v1/products) e permite visualizar, navegar e pesquisar produtos de forma paginada.

## Funcionalidades

- Listagem paginada de produtos
- Visualização de detalhes do produto
- Carrossel de imagens do produto
- Mensagens de carregamento e erro
- Layout responsivo com Material-UI

## Como rodar o projeto

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/seu-usuario/seu-repo.git
   cd desafio-localiza-labs-rafael-maus
   ```
2. **Instale as dependências:**
   ```bash
   npm install
   ```
3. **Rode o projeto:**
   ```bash
   npm start
   ```
   O projeto estará disponível em [http://localhost:3000](http://localhost:3000).

## Tecnologias utilizadas

- React
- TypeScript
- Material-UI
- Axios

## Considerações finais

Este projeto foi desenvolvido como parte do desafio LocalizaLabs e tem como objetivo demonstrar habilidades em desenvolvimento front-end com React e TypeScript. O catálogo de produtos é consumido da API pública EscuelaJS Products e apresenta uma listagem paginada, com possibilidade de visualização dos detalhes e imagens dos produtos.

A aplicação foi desenvolvida utilizando as melhores práticas de programação e está estruturada de forma a facilitar a manutenção e evolução futura.
